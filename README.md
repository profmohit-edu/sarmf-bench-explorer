# SARMF-Bench Explorer

SARMF-Bench Explorer is a smart contract vulnerability benchmarking system developed for analyzing and comparing security issues across multiple detection tools.

## Features

- Smart contract vulnerability dataset
- Filtering by vulnerability type and severity
- Tool-wise analysis (Slither, Mythril, Oyente)
- Interactive dashboard interface

## Objective

To provide a reproducible and scalable benchmarking environment for evaluating smart contract security tools and vulnerability patterns.
